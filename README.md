# Ngôn ngữ sử dụng
    * Nodejs
# Hướng dẫn cài đặt
    1. Cài đặt nodejs
    2. Cài đặt mongoDB hoặc có thể sử dụng mongo online
    3. Khi clone project về thực hiện các bước
        * Config lại connection string của database hoặc có thể sử dụng link database online
        * Thực hiện câu lệnh: npm i
        * Run server: npm start
        * Truy cập đường dẫn: http://localhost:3000/
